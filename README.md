# LucidMerged Public Export

This repository is the public target for the generated, self-hostable LucidMerged export.

The full source tree is synchronized from the private source-of-truth repository through the `sync-open-source.yml` export workflow after OSS boundary, secret, license, install, typecheck, build, and test gates pass.

Official desktop/mobile app source, signing credentials, native release workflows, store metadata, and private planning artifacts are intentionally excluded.
