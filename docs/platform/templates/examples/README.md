# Template Pack Examples

This folder contains importable `LucidPackManifest` examples for creators.

Validate an example before importing it:

```bash
npm run templates:validate -- docs/platform/templates/examples/prospect-intelligence-pack.json
```

Import it from `/templates` with **Import Pack JSON**. Imported Packs are workspace-private by default; use **Submit for review** from the same creator card when the Pack has validation, smoke, and quality evidence. Submission creates a Pack-native marketplace review record; it does not publish globally until Lucid approves it.

## Example Packs

- `prospect-intelligence-pack.json`: a deploy-compatible sales agent template with conservative memory, approval-required write tools, composition metadata, and setup-safe plugin declarations.
